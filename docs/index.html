Payments API Reference

This is a payments API design. The API will utilise the go-json-rest web framework backed by gorm, connecting to a postgres database.

API Endpoint
http://localhost/
Request Content-Types: application/json
Response Content-Types: application/json

A list of monies to be deducted from somewhere.

Money
